# SmartEventCalendar

A modern, interactive web calendar built with JavaScript. Designed to help you manage events efficiently while staying motivated and tracking your progress.

## Features
- **Add / Edit Events** – Easily create events with time, priority, and emoji support.
- **Mark as Done** – Check off events; completed events visually scale and strike through.
- **Drag & Drop** – Move events between days effortlessly.
- **Progress Tracking** – Dynamic progress bar showing completion of daily events.
- **Daily Motivational Quotes** – Get an inspiring quote for each selected day.
- **Animations** – Smooth transitions for adding, removing, and completing events.
- **Dark Mode** – Toggle between light and dark themes for better readability.
- **Export to iPhone** – Export selected day's events as an `.ics` file compatible with Apple Calendar.
- **Undo / Redo** – Revert or restore changes with unlimited flexibility.

## Built With
- HTML5 & CSS3
- JavaScript (ES6)
